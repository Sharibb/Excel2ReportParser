How Phase 3 works:
  1. Upload Three Files:
    • Excel file with vulnerability data
    • Word template for report format
    • ZIP file containing PoC screenshot folders
  2. Automatic Processing:
    • System extracts the ZIP file
    • Matches PoC folders to vulnerability IDs (C1→C1/, H1→H1/)
    • Finds screenshot files (1.png, 2.png, etc.) in each folder
  3. Report Generation:
    • Populates Word template with Excel data
    • Automatically inserts PoC images for each vulnerability
    • Preserves all template formatting
  4. Download:
    • Get your complete professional report
    • All vulnerabilities with screenshots included
Tip: Make sure your PoC folder names exactly match the vulnerability IDs in your Excel file (case-insensitive).
